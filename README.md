# PDS 3D Component

一个基于React和Three.js的3D模型查看器组件库。

## 功能特性

- 🎯 **GLTF模型加载和显示**
- 📦 **包围盒辅助工具** - 可视化模型尺寸和位置信息
- 🎮 **交互式控制** - 相机旋转、缩放、平移
- 💡 **智能灯光系统**
- 🏷️ **设备标签系统**
- ⚡ **性能优化**

## 包围盒辅助功能

包围盒辅助工具可以帮助您：

1. **可视化模型边界** - 显示绿色线框表示模型的包围盒
2. **定位模型中心** - 显示红色球体标记模型的几何中心
3. **分析模型尺寸** - 在控制台输出详细的尺寸和位置信息
4. **优化相机视角** - 提供推荐的相机距离

### 使用方法

```tsx
<GLTFViewer
  modelList={[yourModel]}
  boundingBoxConfig={{
    enabled: true,           // 启用包围盒显示
    showBox: true,          // 显示包围盒线框
    showCenter: true,       // 显示中心点
    boxColor: "#00ff00",    // 包围盒颜色（绿色）
    centerColor: "#ff0000", // 中心点颜色（红色）
    centerSize: 0.1,        // 中心点球体大小
    lineWidth: 2,           // 线条宽度
  }}
/>
```

### 控制面板

在开发模式下，您可以通过Leva控制面板实时调整：

- ✅ **启用包围盒显示** - 总开关
- 📦 **显示包围盒** - 切换包围盒线框显示
- 🎯 **显示中心点** - 切换中心点显示
- 🎨 **包围盒颜色** - 调整线框颜色
- 🔴 **中心点颜色** - 调整中心点颜色
- 📏 **中心点大小** - 调整中心点球体尺寸
